如何使用

1° 下载安装

前往 Releases 下载符合你设备的 APK：
• 一般设备 (Android 8.0+) 下载 Minsdk26
• Android 10+ 下载 Minsdk29
• Android 15+ 下载 Minsdk35
• 如果不清楚，请直接下载 universal 通用版。

2° 配置 API

按照提示填入模型提供商的 URL、Key 以及 模型名称。

3° 开始使用

回到主页，点击下方的输入框，开始尽情使用吧！
